# Diamond Square Resident App

A resident-only Expo/React Native prototype for Diamond Square apartment building management.

## Features included

- Resident login and registration flow with an admin approval/OTP concept.
- Private flat directory where each resident can view only their own flat details.
- Admin/committee notice board for announcements.
- View-only maintenance payment reminders with due date messaging.
- Important contacts for committee members and service providers.
- Birthday, anniversary, maintenance, and complaint-status reminder notifications represented in the resident dashboard.
- Complaint submission with local status tracking.

## Run locally

```bash
npm install
npm run start
```

Use Expo Go or an emulator to open the app.
